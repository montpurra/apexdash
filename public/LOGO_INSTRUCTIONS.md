# Logo Setup Instructions

## Required Files

Place your logo image files in this `public` folder:

1. **logo.png** - Main logo image (used in sidebar and challenges page)
   - Recommended size: 48x48px to 128x128px
   - Format: PNG with transparent background (preferred) or solid background
   - This will be used in the sidebar and on the challenges page

2. **favicon.ico** - Browser favicon
   - Standard size: 16x16px, 32x32px, or 48x48px
   - Format: ICO file
   - This appears in browser tabs

## File Names

The code expects these exact file names:
- `/public/logo.png` - Main logo
- `/public/favicon.ico` - Favicon

## Alternative Formats

If you have a different format:
- SVG: Can be used as `logo.svg` (update code to use `.svg` extension)
- JPG: Can be used as `logo.jpg` (update code to use `.jpg` extension)

After adding the files, the logo will automatically appear in:
- Sidebar (top left)
- Challenges page header
- Browser tab (favicon)
